Explicando Form

Form é um container para elementos de input, usado para coletar inputs de usuário. Normalmente é enviado à um servidor ou outro lugar para ser processado.

As tags "input" definem como o usuário vai dar seu input, seja por texto, selecionando caixas, mandando um arquivo, etc

As tags "label" servem para acessibilidade, visto que são mais facilmente lidas por leitores de tela e quando clicadas botam a caixa de input em foco

Método POST

POST é um método HTML

Ele serve para enviar dados por meio de uma requisição HTTP

Diferentemente do método GET (que envia no cabeçalho), o POST manda esses dados no corpo da requisição
